## 🚀 快速启动指南

### 开发环境启动
```bash
# Windows
双击 start.bat

# Linux/Mac  
./start.sh

# 手动启动
cd backend && npm install && npm run dev
cd frontend && npm install && npm run dev
```

### 生产环境部署
```bash
# Docker 一键部署
docker-compose -f docker-compose.prod.yml up -d

# 传统部署
sudo ./scripts/deploy-production.sh
```
